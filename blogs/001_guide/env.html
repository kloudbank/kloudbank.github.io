    Dev. Environment

    noisonnoiton 3/15/2021 development

    # IDE

    # 1. VS Code

    • Visual Studio Code 설치
      https://code.visualstudio.com/download (opens new window)

    # 2. Java 개발 환경

    • OpenJDK 11 설치
      https://jdk.java.net/archive/ (opens new window)
    • VS Code Java Extension Pack 설치
      https://code.visualstudio.com/docs/java/extensions (opens new window)
      • VS Code Java Development Tutorial
        https://code.visualstudio.com/docs/java/java-tutorial (opens new window)

    # 3. Spring Boot 개발 환경

    • VS Code Spring Boot Extension Pack 설치하여, 아래 extension 일괄 적용
      • Spring Boot Tools
      • Spring Initializr
      • Spring Boot Dashboard
        https://code.visualstudio.com/docs/java/java-spring-boot (opens new window)

    # Container 개발 환경

    # 1. Docker, kubectl

    • Dockerhub 계정 생성 (계정 없을 경우)
      https://hub.docker.com/ (opens new window)
    • Docker Desktop 설치
      https://docs.docker.com/desktop/ (opens new window)

    # 2. Kubernetes Object 관리

    • kustomize 설치
      https://kubectl.docs.kubernetes.io/installation/kustomize/ (opens new window)

    • kustomize를 이용한 k8s object 선언형 관리
      https://kubernetes.io/ko/docs/tasks/manage-kubernetes-objects/kustomization/ (opens new window)

      • kustomization 파일을 포함하는 디렉터리 내의 리소스 확인
    kubectl kustomize <kustomization_directory>
    1
    • 리소스를 적용하려면 kubectl apply를 --kustomize 또는 -k 플래그와 함께 실행
    kubectl apply -k <kustomization_directory>
    1

    # 3. AWS 및 EKS Config

    • AWS 구성 가이드 참고
      https://htdp1.github.io/aws/#aws-configure (opens new window)
