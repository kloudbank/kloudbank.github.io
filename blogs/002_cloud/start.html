    AWS 및 EKS 개발 환경 설정.

    # Prerequisite

    • aws-cli 설치
    • AWS Access Key 생성
    • 기본 Region은 Seoul (ap-northeast-2)

    # Create kubeconfig

    • kubeconfig 생성 후, kube-system aws-auth configmap에 mapUser 추가 필요
    aws eks update-kubeconfig \
  --region ap-northeast-2 \
  --name <cluster name> \
  --profile <username>
