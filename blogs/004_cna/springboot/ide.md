::: tip
export extension list
``` sh
code --list-extensions > extension.txt
```
import extension list
```sh
# powershell
type extension.txt |  % { "code --install-extension $_" }
#linux
cat extension.txt | xargs -L 1 echo code --install-extension
```
:::
