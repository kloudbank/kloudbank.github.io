    본 가이드에서는 Visual Studio Code (opens new window)를 활용하여 설명하고 있어, Springboot을 위한 설정과 환경변수 내용을 설명 합니다.

    # Step 1: JDK 11 설치

    • jdk-11.0.11-x64 MSI : redhat 가입 필요. https://developers.redhat.com/products/openjdk/download
    • JDK_HOME 등록
    • JAVA_HOME 등록

    # Step 2: Maven 설치

    • maven 설치 : download(https://mirror.navercorp.com/apache/maven/maven-3/3.8.1/binaries/apache-maven-3.8.1-bin.zip)
    • unzip : c:\maven
    • 환경변수 설정: MAVEN_HOME 등록
    • Path 추가 :%MAVEN_HOME%\bin
    • SSL Invalidate 설정 : [TODO]

    • Project Manager for Java (opens new window)
    • Visual Studio IntelliCode (opens new window)
    • Spring Boot Tools (opens new window)
    • Spring Initializr Java Support (opens new window)
    • Spring Boot Dashboard (opens new window)
